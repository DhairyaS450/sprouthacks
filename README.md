# Gemini Chat Application

This project was created during the SproutHacks 2025 hackathon in 36 hours so don't judge lmao

## Gemini Chat Interface

This is a simple command-line chat interface for Google's Gemini AI model.

### Setup

1. Make sure you have Node.js installed on your system
2. Create a `.env` file in the root directory with your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
3. Install dependencies:
   ```
   npm install
   ```

### Usage

Run the chat application:
```
node gemini-chat.js
```

- Type your messages and press Enter to chat with Gemini
- Type 'exit' to end the conversation

### Features

- Interactive command-line chat interface
- Maintains conversation history for context
- Simple and easy to use
